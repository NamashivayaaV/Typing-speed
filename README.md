# Typing Speed Test

A simple command-line typing speed test application built in Python. Test your typing speed and accuracy with different difficulty levels.

## Features

- Choose between Easy and Hard difficulty levels
- Random text samples for each difficulty
- Calculates Words Per Minute (WPM)
- Measures typing accuracy percentage
- Timed typing session

## How to Run

1. Ensure you have Python installed on your system.
2. Clone the repository or download the files.
3. Run the main script:

```bash
python typing_test.py
```

4. Follow the on-screen instructions to select difficulty and start typing.

## Requirements

- Python 3.x
